MailsCool — Privacy Policy

Last updated: March 15, 2026

MailsCool ("the Extension") is a browser extension that provides disposable temporary email addresses. This Privacy Policy explains what data the Extension accesses, how it is used, and how it is stored.

Summary: MailsCool does not collect, transmit, or sell any personal user data. All data is stored locally on your device.

1. Data We Collect

The Extension does not collect any personal information from users. The only data stored is:

  • Temporary mailbox credentials — A randomly generated email address and password created by the email service, stored locally via chrome.storage.local to maintain your session.
  • API configuration — An optional server URL and API key you may provide to connect to a self-hosted email service.

This data is stored entirely on your local device and is never transmitted to any third party.

2. How We Use Data

The locally stored data is used exclusively to:

  • Maintain your current temporary mailbox session across popup opens/closes.
  • Authenticate with the email service API to create mailboxes and retrieve incoming emails.
  • Auto-fill temporary email addresses and verification codes into web page input fields.
  • Display desktop notifications when new emails arrive.

3. Data Sharing

We do not:

  • Sell or transfer user data to third parties.
  • Use data for advertising, analytics, or tracking purposes.
  • Use data for credit determination or lending purposes.
  • Collect browsing history, personal information, or user activity.

4. Permissions

The Extension requests the following browser permissions, used solely for its core functionality:

  • storage — Save temporary mailbox credentials and API settings locally.
  • activeTab — Send verification codes to the active tab for auto-filling.
  • notifications — Alert you when new emails arrive.
  • tabs — Identify the active tab to deliver auto-fill messages.
  • Host permissions (*://*/*) — Inject the content script that detects email input fields and provides the in-page auto-fill widget on any website.

5. Remote Code

The Extension does not use any remote code. All JavaScript is bundled locally within the extension package. The Extension only makes API calls to the configured email service to create mailboxes and retrieve emails.

6. Data Retention

Temporary mailbox data is stored in chrome.storage.local until you create a new mailbox (which replaces the previous data) or uninstall the Extension. Uninstalling the Extension automatically clears all stored data.

7. Changes to This Policy

We may update this Privacy Policy from time to time. Any changes will be reflected on this page with an updated "Last updated" date.

8. Contact

If you have questions about this Privacy Policy, please contact us at support@mails.cool.
